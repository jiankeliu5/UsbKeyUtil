======================================================
    目录 & 说明
======================================================

 Java : ePass1000ND的Java API接口目录，jdk版本为1.6.0_13；

 FT_ND_API.dll 和 FT_ND_API.lib : ePass1000ND的私有接口库文件；

 FT_ND_MOD.dll : ePass1000ND API 全功能控件；

 FT_ND_SC.dll : ePass1000ND API 安全功能控件，主要用于网页；

 JePs1knd.dll : ePass1000ND JNI 接口；

 FT_ND_FULL.dll : ePass1000ND API 功能齐备的安全功能控件，主要用于网页。
